# ולנטינה נחשונוב · אתר מעצבת בגדים

אתר תדמית וחנות עבור מעצבת הבגדים **ולנטינה נחשונוב**. עוצב בפלטת נייבי · שמנת · זהב, במראה עתידני, חלק ומודרני.

## ✦ תכונות עיקריות

- **חנות אונליין** — כרטיסיות פריטים עם תווית מחיר, בחירת מידות, פילטרים והזמנה בוואטסאפ.
- **השכרת שמלות ערב** — כל שמלה עם כרטיסייה משלה (מחיר ליום, פיקדון, מידות, זמינות).
- **בלוג אופנה** — 9 מאמרים (300+ מילים), מוצגים 3 בכל פעם עם החלפה אוטומטית ובחירה ידנית.
- **תיק עבודות** — העלאת תמונות בגרירה/לחיצה.
- **טופס יצירת קשר** המקושר ישירות לוואטסאפ.
- **כפתורים צפים** — וואטסאפ והתקשרות בצד ימין.
- **תפריט נגישות צף** בצד שמאל (הגדלת/הקטנת טקסט, ניגודיות, גווני אפור, הדגשת קישורים, גופן קריא, עצירת אנימציה, סמן גדול) — עם שמירת העדפות.
- **הצהרת נגישות, מדיניות אתר ותקנון ומדיניות פרטיות.**
- **וידאו HERO** וסרטוני קולקציה משולבים לאורך האתר, אנימציות מרובות וחשיפה בגלילה.
- תמיכה מלאה ב־RTL ועיצוב רספונסיבי.

## 🛠️ טכנולוגיות

HTML5 · CSS3 · JavaScript (Vanilla) — ללא תלויות חיצוניות.
